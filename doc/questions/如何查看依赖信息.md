# 如何查看依赖信息？

```
#跳过测试编译打包
mvn clean package -DskipTests
#查看依赖树中是否包含org.apache.httpcomponents
mvn -q dependency:tree -Dincludes=org.apache.httpcomponents
#查看有效POM中是否有httpclient5
mvn help:effective-pom | grep httpclient5
#查看jar包中是否有TlsSocketStrategy类
jar tf feng-gateway3.jar | grep TlsSocketStrategy
```
